Flutter & Dart: Phát Triển Mobile App Cross-Platform

05 Dec 2024 Flutter 19 phút đọc
← Quay lại Blog

Flutter Là Gì?

Flutter là open-source UI framework được phát triển bởi Google. Nó cho phép bạn xây dựng native mobile applications cho iOS và Android từ một codebase duy nhất bằng Dart language.

Tại Sao Chọn Flutter?

  • Cross-platform: Một codebase cho iOS và Android
  • Fast development: Hot reload giúp development nhanh chóng
  • Beautiful UI: Rich widgets và animations
  • High performance: Compiled nên performance rất tốt
  • Large community: Cộng đồng rất lớn với nhiều packages

Dart Language

Dart là ngôn ngữ được tối ưu cho Flutter development:

// Variables var name = 'John'; String email = 'john@example.com'; int age = 25; double height = 1.75; bool isStudent = true; // Functions String greet(String name) { return 'Hello, $name!'; } // Arrow functions int add(int a, int b) => a + b;

Widgets

Widgets là building blocks của Flutter apps. Mọi thứ là widgets:

import 'package:flutter/material.dart'; void main() { runApp(MyApp()); } class MyApp extends StatelessWidget { @override Widget build(BuildContext context) { return MaterialApp( home: Scaffold( appBar: AppBar(title: Text('My App')), body: Center( child: Column( mainAxisAlignment: MainAxisAlignment.center, children: [ Text('Hello World'), SizedBox(height: 20), ElevatedButton( onPressed: () {}, child: Text('Click me'), ), ], ), ), ), ); } }

Stateful vs Stateless Widgets

Stateless Widget

Không thay đổi state:

class MyButton extends StatelessWidget { @override Widget build(BuildContext context) { return ElevatedButton( onPressed: () {}, child: Text('Click'), ); } }

Stateful Widget

Có thể change state:

class Counter extends StatefulWidget { @override State<Counter> createState() => _CounterState(); } class _CounterState extends State<Counter> { int count = 0; @override Widget build(BuildContext context) { return Column( children: [ Text('Count: $count'), ElevatedButton( onPressed: () { setState(() { count++; }); }, child: Text('Increment'), ), ], ); } }

State Management

Quản lý state phức tạp với Provider hoặc Riverpod:

// Provider final counterProvider = StateNotifierProvider<CounterNotifier, int>((ref) { return CounterNotifier(); }); class CounterNotifier extends StateNotifier<int> { CounterNotifier() : super(0); void increment() { state++; } }

Navigation

// Push to new screen Navigator.push( context, MaterialPageRoute(builder: (context) => SecondScreen()), ); // Pop back Navigator.pop(context); // Named routes MaterialApp( routes: { '/': (context) => HomeScreen(), '/details': (context) => DetailsScreen(), }, onGenerateRoute: (settings) { if (settings.name == '/user') { final id = settings.arguments; return MaterialPageRoute( builder: (context) => UserScreen(id: id), ); } }, )

HTTP Requests

import 'package:http/http.dart' as http; Future<List> fetchUsers() async { final response = await http.get( Uri.parse('https://api.example.com/users'), ); if (response.statusCode == 200) { return jsonDecode(response.body); } else { throw Exception('Failed to load users'); } }

Best Practices

  • Sử dụng const constructors khi possible
  • Implement proper state management
  • Keep widgets focused and small
  • Use proper error handling
  • Optimize performance sử dụng appropriate widgets
  • Write tests cho business logic

Kết Luận

Flutter là framework tuyệt vời cho mobile development. Với Dart language, hot reload, và rich widget library, bạn có thể build beautiful mobile apps nhanh chóng và hiệu quả.
