Node.js & Express: Backend JavaScript Full Power

08 Dec 2024 Node.js 18 phút đọc
← Quay lại Blog

Node.js Là Gì?

Node.js là JavaScript runtime environment cho phép chạy JavaScript ở server side. Nó được xây dựng trên V8 engine của Google và cho phép xây dựng scalable backend applications.

Express Framework

Express là web application framework phổ biến nhất cho Node.js. Nó cung cấp tools để xây dựng REST APIs và web servers dễ dàng.

Cài Đặt & Setup

npm init -y npm install express npm install --save-dev nodemon

Server Đầu Tiên

const express = require('express'); const app = express(); app.use(express.json()); app.get('/', (req, res) => { res.json({ message: 'Hello World' }); }); const PORT = process.env.PORT || 3000; app.listen(PORT, () => { console.log(`Server running on port ${PORT}`); });

Routing

// GET request app.get('/api/users', (req, res) => { res.json([{ id: 1, name: 'John' }]); }); // POST request app.post('/api/users', (req, res) => { const { name, email } = req.body; res.status(201).json({ id: 1, name, email }); }); // PUT request app.put('/api/users/:id', (req, res) => { res.json({ id: req.params.id, ...req.body }); }); // DELETE request app.delete('/api/users/:id', (req, res) => { res.json({ message: 'User deleted' }); });

Middleware

Middleware là functions xử lý requests trước khi đến routes:

// Logging middleware app.use((req, res, next) => { console.log(`${req.method} ${req.path}`); next(); }); // Authentication middleware const authenticate = (req, res, next) => { const token = req.headers.authorization; if(token) { next(); } else { res.status(401).json({ error: 'Unauthorized' }); } }; app.get('/api/protected', authenticate, (req, res) => { res.json({ data: 'Protected data' }); });

Error Handling

// Route not found app.use((req, res) => { res.status(404).json({ error: 'Not found' }); }); // Error handler app.use((err, req, res, next) => { console.error(err.stack); res.status(500).json({ error: 'Internal server error' }); });

Database Integration

Sử dụng MongoDB hoặc SQL databases:

const mongoose = require('mongoose'); mongoose.connect('mongodb://localhost:27017/myapp'); const userSchema = new mongoose.Schema({ name: String, email: String, createdAt: { type: Date, default: Date.now } }); const User = mongoose.model('User', userSchema); // Create const newUser = await User.create({ name: 'John', email: 'john@example.com' }); // Read const users = await User.find(); const user = await User.findById(id); // Update await User.findByIdAndUpdate(id, { name: 'Jane' }); // Delete await User.findByIdAndDelete(id);

Authentication

Sử dụng JWT cho authentication:

const jwt = require('jsonwebtoken'); // Generate token const token = jwt.sign( { userId: user._id }, 'your-secret-key', { expiresIn: '1h' } ); // Verify token const decoded = jwt.verify(token, 'your-secret-key');

Best Practices

  • Sử dụng environment variables cho sensitive data
  • Implement proper error handling và logging
  • Validate input data từ client
  • Sử dụng middleware cho cross-cutting concerns
  • Organize code theo modules
  • Implement authentication & authorization
  • Test APIs với postman hoặc unit tests

Kết Luận

Node.js với Express là powerful combination cho building modern backend applications. Với JavaScript trên cả client và server, bạn có thể share code và concepts, tăng productivity.
