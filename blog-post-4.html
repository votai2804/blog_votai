Bootstrap 5 & jQuery: Xây Dựng Giao Diện Web Động

12 Dec 2024 Frontend 17 phút đọc
← Quay lại Blog

Bootstrap 5 Là Gì?

Bootstrap 5 là framework CSS phổ biến nhất thế giới, giúp bạn xây dựng responsive websites nhanh chóng. Nó cung cấp các components, utilities, và grid system sẵn sàng sử dụng.

Grid System

Bootstrap 5 sử dụng CSS Grid để tạo layout responsive:

<div class="container"> <div class="row"> <div class="col-md-6">50% width on medium screens</div> <div class="col-md-6">50% width on medium screens</div> </div> </div>

Components

Bootstrap cung cấp các components đã được styled:

  • Buttons với các variants (primary, secondary, success, danger)
  • Cards để hiển thị content
  • Modals cho dialogs
  • Navbars, breadcrumbs, pagination
  • Forms, tables, alerts
  • Carousels, spinners, toasts

Utilities

Bootstrap utilities giúp styling nhanh chóng:

<div class="mt-5 mb-3 p-4 bg-light rounded"> <h3 class="text-center text-primary">Tiêu đề</h3> <p class="text-muted">Đây là một paragraph</p> </div>

jQuery Là Gì?

jQuery là một JavaScript library giúp làm việc với DOM dễ dàng hơn:

// jQuery DOM selection $('#myElement'); $('.myClass'); $('div p'); // DOM manipulation $('#btn').text('Click me'); $('#form').hide(); $('.item').addClass('active');

AJAX với jQuery

AJAX cho phép load dữ liệu mà không cần reload page:

$.ajax({ url: '/api/products', type: 'GET', success: function(data) { console.log(data); // Render data }, error: function(error) { console.log('Error:', error); } });

Event Handling

jQuery làm event handling đơn giản:

$('#submitBtn').on('click', function() { const name = $('#nameInput').val(); if(name) { alert('Hello ' + name); } }); $('.deleteBtn').on('click', function(e) { e.preventDefault(); if(confirm('Are you sure?')) { // Delete logic } });

Best Practices

  • Sử dụng Bootstrap classes trước khi viết CSS custom
  • Cache jQuery selectors trong variables
  • Sử dụng event delegation cho elements được thêm động
  • Minimize DOM manipulation trong loops
  • Sử dụng .on() thay vì .click(), .change()
  • Test responsiveness trên các devices khác nhau

Kết Luận

Bootstrap 5 và jQuery là combination tuyệt vời để xây dựng web interfaces động và responsive. Hãy master các tools này để tăng productivity trong frontend development.
