Về Tôi

Lập Trình Viên - Công Nghệ Phần Mềm

Tôi là một lập trình viên có chuyên ngành Công nghệ phần mềm từ ĐH Công nghệ TP.HCM (HUTECH). Với tư duy hệ thống vững chắc và khả năng tự học tốt, tôi luôn cập nhật các Framework và công nghệ mới nhất như .NET và Node.js để phát triển giải pháp phần mềm hiệu quả.

Năng Lực Kỹ Thuật Chính

  • Thành thạo nhiều ngôn ngữ lập trình: C#, Java, JavaScript, Dart
  • Chuyên sâu Backend: ASP.NET Core Web API và thiết kế hệ thống
  • Database: SQL Server 2022, Entity Framework Core
  • Frontend: Bootstrap 5, jQuery/AJAX, giao diện Web Admin
  • Quản lý mã nguồn: Git Flow, Postman API Testing

Điểm Mạnh & Phong Cách Làm Việc

Tôi có tư duy hệ thống vững chắc, khả năng tự học tốt thông qua các dự án thực tế và tinh thần cầu tiến trong việc cập nhật công nghệ. Làm việc hiệu quả trong team, sẵn sàng chia sẻ kiến thức và luôn mở rộng kiến thức kỹ thuật.

Về Portfolio

Trang web cá nhân để giới thiệu về kinh nghiệm, kỹ năng và các dự án của tôi trong lĩnh vực phần mềm.
