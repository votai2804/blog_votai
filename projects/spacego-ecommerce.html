Tổng Quan Dự Án

Xây dựng các chức năng bán hàng, quản lý sản phẩm và thanh toán cho cửa hàng thiết bị di động. Hoàn thiện MVP trong 10 tuần với chức năng đặt hàng và thanh toán mô phỏng hoạt động ổn định.

Nhiệm Vụ Chính

  • Phát triển hệ thống quản lý sản phẩm, giỏ hàng và đơn hàng bằng ASP.NET MVC.
  • Thiết kế schema SQL Server, triển khai stored procedures và tối ưu query.
  • Đảm bảo responsive front-end và tích hợp gateway thanh toán (sandbox) cho thử nghiệm.

Công Nghệ

ASP.NET MVC C# SQL Server jQuery Bootstrap
