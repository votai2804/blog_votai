Face Recognition Attendance System

Developer

Jul 2023

Tổng Quan Dự Án

Hệ thống chấm công bằng nhận diện gương mặt, viết bằng Python. Thu nhận và mã hóa khuôn mặt, so khớp và lưu nhật ký chấm công vào SQL Server; có giao diện quản trị đơn giản để duyệt lịch sử.

Mô Tả Chi Tiết

Hệ thống chấm công tự động sử dụng công nghệ nhận diện gương mặt để thay thế phương pháp chấm công truyền thống. Ứng dụng có khả năng:

  • Thu nhận hình ảnh khuôn mặt từ webcam
  • Mã hóa và lưu trữ dữ liệu khuôn mặt an toàn
  • So khớp khuôn mặt real-time với cơ sở dữ liệu
  • Ghi nhận thời gian chấm công tự động
  • Quản trị lịch sử chấm công qua giao diện web

Công Nghệ

Python OpenCV face-recognition SQL Server

Tính Năng Chính

  • Nhận Diện Gương Mặt: Sử dụng OpenCV và face-recognition library
  • Mã Hóa Khuôn Mặt: Chuyển đổi khuôn mặt thành vector 128-chiều
  • So Khớp Thời Gian Thực: Nhận diện người dùng trong vòng <100ms
  • Ghi Nhật Ký: Lưu thời gian, người dùng, mức độ chính xác vào SQL Server
  • Giao Diện Quản Trị: Duyệt lịch sử, thống kê tham dự, export báo cáo
  • Đăng Ký Khuôn Mặt: Thêm người dùng mới, cập nhật dữ liệu khuôn mặt

Chi Tiết Kỹ Thuật

Quy Trình Hoạt Động:

  1. Cấp phát webcam và thu nhận frame video
  2. Phát hiện khuôn mặt bằng OpenCV (Haar Cascade hoặc SSD)
  3. Trích xuất embedding vector 128-chiều từ khuôn mặt
  4. So khớp với embedding trong cơ sở dữ liệu (Euclidean distance)
  5. Nếu match (distance < threshold), ghi nhân và lưu nhật ký
  6. Cập nhật giao diện quản trị cho người dùng

Bảo Mật Dữ Liệu:

  • Không lưu trữ ảnh gốc, chỉ lưu embedding vector
  • Mã hóa dữ liệu nhạy cảm trong SQL Server
  • Xác thực người dùng trước khi truy cập quản trị
  • Ghi log mọi thao tác trên hệ thống
